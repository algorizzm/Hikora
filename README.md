# Hikora
### Hiking Event Booking Application for Cebu

Hikora is a mobile application designed to help hikers discover, join, and organize hiking events across Cebu. The platform connects hikers, guides, and event organizers in a centralized system where users can browse trails, book hikes, and manage outdoor adventures easily.

This project is being developed using **Android Studio with Kotlin** and follows an **Agile development process with sprint cycles from March to May**.

---

# Project Objectives

The main objective of Hikora is to create a mobile platform that allows users to:

- Discover hiking trails in Cebu
- Book hiking events
- Organize hiking activities
- Connect hikers with guides and organizers
- Manage bookings and event participation efficiently

---

# User Roles

Hikora supports multiple user roles within the system:

### Hiker
Regular users who want to browse trails and book hiking events.

### Guide
Experienced hikers who assist during organized hiking events.

### Organizer
Users who create and manage hiking events.

### Administrator
System managers responsible for monitoring users and maintaining the platform.

---

# System Modules

## 1. Registration Module
Allows new users to create accounts in the system.

**Features**
- Register new user
- Account creation for Hiker, Guide, Organizer, or Admin

---

## 2. Profile Management Module
Allows users to manage their personal information and account settings.

**Features**
- Update profile information
- Manage account settings
- Edit contact details
- Update profile picture (future feature)

---

## 3. User Management Module (Admin Only)
Allows administrators to control and manage platform users.

**Features**
- View users
- Update user information
- Manage user status (active/suspended)
- Delete user accounts

---

## 4. Hiking Event Management Module
Allows organizers to create and manage hiking events.

**Features**
- Create hiking event
- Update hiking event
- Cancel hiking event
- View event listings
- Set maximum participants
- Add event description and schedule

---

## 5. Booking Management Module
Allows hikers to reserve slots for hiking events.

**Features**
- Book hiking event
- View bookings
- Cancel bookings
- Track participation status

---

## 6. Trail Information Management Module
Provides detailed information about hiking locations in Cebu.

**Features**
- View trail catalog
- View trail difficulty level
- View trail location and description
- Manage trail information (Organizer/Admin)

---

# Future Features

The following features may be implemented in later development sprints:

- GPS trail navigation
- Interactive map integration
- Event notifications
- Weather alerts for hikes
- Guide ratings and reviews
- Photo sharing for hikes
- Payment integration for paid events
